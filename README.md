# TF-IDF Search System

**Repository**: https://github.com/alexgeen896-stack/tfidf_search_system_Samarets_Sergey

## Цель проекта
Построение системы поиска по текстовой базе данных с использованием TF-IDF и косинусного сходства.

## Структура проекта
\\\
tfidf_search_system/
├── data/
│   ├── raw/           # Исходные CSV файлы (текстовая база)
│   └── processed/     # Обработанные данные и TF-IDF индексы
├── src/
│   ├── prepare.py     # Подготовка данных, очистка текста
│   ├── index.py       # Построение TF-IDF матрицы
│   └── search.py      # Поиск по запросу
├── dvc.yaml          # Описание пайплайна
├── params.yaml       # Параметры TF-IDF
├── requirements.txt  # Зависимости
└── README.md         # Документация
\\\

## Описание системы
1. **Подготовка данных** - загрузка CSV, очистка текста (stemming, stop-words)
2. **Построение TF-IDF** - создание векторного представления документов
3. **Поиск** - вычисление косинусного сходства между запросом и документами
4. **Ранжирование** - возврат наиболее релевантных документов

## Технологии
- **Python** - основной язык
- **Scikit-learn** - TF-IDF векторизация
- **Pandas** - обработка CSV данных
- **NLTK/Spacy** - обработка естественного языка
- **DVC** - управление данными и пайплайнами
- **MLflow** (опционально) - трекинг экспериментов

## Особенности
- Поддержка русского и английского языков
- Настройка параметров TF-IDF через params.yaml
- Автоматизация пайплайна через DVC
- Воспроизводимость экспериментов
