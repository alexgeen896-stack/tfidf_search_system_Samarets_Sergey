# TF-IDF Search System

**Repository**: https://github.com/alexgeen896-stack/tfidf_search_system_Samarets_Sergey

## Цель проекта
Построение системы поиска по медицинским товарам с использованием TF-IDF и косинусного сходства. Система позволяет искать товары по названиям, описаниям и брендам.

## Структура проекта

<pre>
tfidf_search_system/
├── data/
│   ├── raw/           # Исходные CSV файлы (offers.csv)
│   └── processed/     # Обработанные данные и TF-IDF индексы
├── src/
│   ├── prepare.py     # Подготовка данных, очистка текста, усиление брендов
│   ├── index.py       # Построение TF-IDF матрицы
│   └── search.py      # Интерактивный поиск по запросам
├── dvc.yaml          # Описание пайплайна
├── params.yaml       # Параметры TF-IDF и поиска
├── requirements.txt  # Зависимости
└── README.md         # Документация
</pre>

## Как запустить

### 1. Клонировать репозиторий
```bash
git clone https://github.com/alexgeen896-stack/tfidf_search_system_Samarets_Sergey.git
cd tfidf_search_system_Samarets_Sergey
```
### Установить зависимости
```bash
pip install -r requirements.txt
```

### 3. Запустить пайплайн подготовки данных
```bash
dvc repro
```

### 4. Запустить поисковую систему
```bash
python src/search.py
```

### Описание системы

## Стадия prepare
Загрузка датасета offers.csv (1833 медицинских товара)

Объединение текстовых колонок: name, description, brand

Очистка текста (приведение к нижнему регистру, удаление пунктуации)

Усиление брендов - повторение названий брендов для увеличения веса в TF-IDF

## Стадия index
Построение TF-IDF матрицы (1833 документов × 5000 признаков)

Использование униграмм и биграмм

Сохранение векторизатора и матрицы для быстрого поиска


## Стадия search
Интерактивный поиск по запросам

Вычисление косинусного сходства между запросом и документами

Ранжирование и вывод топ-10 результатов

Поддержка поиска по брендам, названиям, описаниям


## Примеры запросов
По названию товара: "перчатки нитриловые", "электроды экг", "гель ультразвуковой"

По бренду: "Archdale", "Mediok", "Benovy", "Safe&Care"

По описанию: "жидкий гель", "одноразовые перчатки", "хирургические инструменты"


## Технологии
Python 3.13 - основной язык

Scikit-learn - TF-IDF векторизация и косинусное сходство

Pandas - обработка CSV данных

DVC (Data Version Control) - управление данными и пайплайнами

YAML - конфигурация параметров

Pickle - сохранение моделей и индексов


## Особенности реализации
Усиление брендов: Названия брендов повторяются несколько раз в тексте для увеличения веса в TF-IDF

Гибкая настройка: Все параметры (min_df, max_features, ngram_range) вынесены в params.yaml

Воспроизводимость: Полный пайплайн автоматизирован через DVC (dvc repro)

Интерактивный поиск: Удобный интерфейс с подсказками и примерами запросов

Обработка русского языка: Поддержка кириллицы, очистка текста с учетом особенностей языка
